# ☕ The Coffee Club

A modern and visually appealing coffee shop website designed to deliver a clean and user-friendly experience. This project focuses on creating an engaging UI with well-structured sections and smooth navigation.

## 🚀 Features
- Responsive and clean layout
- Attractive landing (hero) section
- Menu showcase with categories
- Image gallery section
- Contact form UI
- Smooth navigation bar

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🎯 Objective
The goal of this project was to improve my frontend development skills by building a real-world website layout with proper structure, design consistency, and usability.
